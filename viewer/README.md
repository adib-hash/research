# AMD Inventory Analysis Viewer

Minimal Vite + React harness for `/research/AMDInventoryAnalysis.jsx`.

## Run

```
cd /Users/adibchoudhury/Documents/claude\ code/research/viewer
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).

To view the NVIDIA file instead, edit `src/main.jsx` to import `NvidiaInventoryAnalysis.jsx`.
